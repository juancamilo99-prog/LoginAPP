Aplicación de escritorio desarrollada en Java con JavaFX que implementa un sistema completo de autenticación y gestión de usuarios con control de acceso por roles.

---

## ✨ Descripción

LoginApp es una aplicación que simula un sistema real de gestión de usuarios, donde existen distintos roles (Administrador y Empleado) con funcionalidades específicas.

El sistema permite gestionar usuarios, autenticar accesos y controlar acciones dentro de la aplicación mediante una interfaz visual dinámica.

---

## 🚀 Funcionalidades principales

### 🔐 Autenticación
- Inicio de sesión con validación de credenciales
- Control de acceso por roles

### 👤 Gestión de usuarios (Admin)
- Alta de usuarios
- Eliminación de usuarios
- Actualización de contraseñas
- Búsqueda por DNI en tiempo real
- Visualización en tabla dinámica

### ⏱️ Gestión de empleados
- Sistema de fichaje con fecha y hora
- Visualización de información del empleado

### 🔄 Extras
- Cierre de sesión
- Persistencia de datos en archivos
- Actualización automática de interfaz (JavaFX)
